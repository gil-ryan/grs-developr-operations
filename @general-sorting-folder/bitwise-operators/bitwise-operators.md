# Bitwise Operators

| Operator  | Description | Example |
|:-:|:-:|:-:|
| & | bitwise AND | Operator copies a bit to the result if it exists in both operands (a & b means 0000 1100) |
| \| | bitwise OR | (a \| b means 0011 1101)
| ^ | bitwise exclusive OR|
| << | shift left |
| >> | shift right|
| ~ | one's complement|
